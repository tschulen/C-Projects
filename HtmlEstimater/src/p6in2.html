Create a matrix operations library - a file containing a set of functions for handling matrices. Matrices are 2D arrays of (typically floating point) numbers. For example:

 Matrices are very common in engineering applications, where the individual elements of the array might, for example, represent voltages on a circuit board.

Your library functions must be able to handle matrices of any size, and must use pointer syntax when manipulating elements (e.g. *(ad + i*iC + j) *= dScalar; , rather than ad[i][j] *= dScalar). Your program should have 3 files - prog7.c, matrix.c, and matrix.h. matrix.c must contain (at least) the following set of functions and employ the given declaration (*adA is a pointer to the 2D array of doubles, iR and iC are the # of rows and columns, respectively):

  • void MatPrint( double *adA, int iR, int iC ) - prints matrix A to the standard output.
  • void MatScalarAdd( double *adA, int iR, int iC, double dScalar ) - adds dScalar to each element of matrix A. For example:
  • void MatScalarMult( double *adA, int iR, int iC, double dScalar ) - multiplies each element of matrix A by dScalar.
  • Double MatSum( double *adA, int iR, int iC) - returns the sum of all of the matrices’ elements.
  • void MatAdd( double *adA, double *adB, double *adC, int iR, int iC) - adds matrix A to matrix B and stores the result in matrix C.
  • void MatMult( double *adA, double *adB, double *adC, int iR, int iCA, int iCB) - multiplies matrix A by matrix B and stores the result in matrix C.
